# Engineering Quick Reference

## Kernstack

- Preact 10, Vite, TypeScript strict
- KoliBri (`@public-ui/preact`)
- UnoCSS fuer Layout, SCSS/BEM fuer visuelle Komponentenstile

## Kernkommandos

```bash
pnpm i
pnpm dev
pnpm lint
pnpm test:e2e
pnpm build
```
